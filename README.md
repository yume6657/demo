# Django + YOLOv8 多目标检测系统

## 项目简介

本项目基于 Django Web 框架与 YOLOv8 深度学习模型，实现了一个多目标图像检测系统。  
用户可通过浏览器上传图片，服务器端完成目标检测并将结果以可视化形式返回。

系统在车牌检测功能的基础上，扩展实现了行人、车辆及红绿灯等多类别目标检测，具有良好的工程扩展性，适用于课程设计与工程实践场景。

---

## 功能说明

系统支持以下检测功能：

- 车牌检测
- 行人检测
- 车辆检测
- 红绿灯检测

用户可在前端页面选择检测类型，系统自动完成对应目标的识别与结果展示。

---

## 技术栈

- 后端框架：Django  
- 目标检测：YOLOv8（Ultralytics）  
- 图像处理：OpenCV  
- 前端技术：HTML / CSS / JavaScript  
- 编程语言：Python 3.x
